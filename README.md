# Installation
Put noPsmWhitelist.skprx into ux0:/tai (or ur0:tai for sd2vita users)
and add an entry under `*KERNEL` for `ux0:/tai/noPsmWhitelist.skprx` (or ur0:/tai for sd2vita!)
reboot the console and enjoy PSM Games on PSTV!
